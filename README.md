> 听得见，写得出，才算真的听懂。

每日英语听写是一个用来练英语听力的小程序。

听力最扎实的练法，还是听写。找一段英文，反复听，听完试着写下来，再回头看原文。能不能听出来，能不能写出来，答案很直接。

很多学习工具会把事情做得很复杂：评分、纠音、分析、建议、报告。那些东西看起来热闹，但真正每天能坚持下来的，往往是更朴素的练习：照着好材料听，照着好声音读，把耳朵磨出来。

## 可以怎么用

- 把英文句子、段落或文章保存成材料。
- 给材料绑定自己的音频，支持 `mp3`、`m4a`、`wav`。
- 按分类整理材料，也可以搜索材料。
- 有音频就直接听，没有音频就先当阅读材料存着。
- 在材料页播放、暂停、继续、从头听、拖动进度、查看原文。
- 用“听写练习”从全部材料、某个分类或练习组里随机播放。
- 默认先听，不急着看原文；需要时再展开对照。

## 为什么叫每日英语听写

因为它适合每天用一点。

不用把学习设计得很宏大。今天听一段，明天再听一段。听不清就从头，听烦了就换材料。你可以用公开材料，也可以把自己喜欢的文章、句子和音频放进去。

读、听、写、跟读，其实可以围绕同一份材料完成。先听，再看；看懂了，再反复听；听熟了，就跟着念。能把材料听熟，比堆很多花哨功能更有用。

## 当前状态

项目当前是微信小程序工程，支持本地使用，也支持 CloudBase 云端资料库接入。

已经具备：

- 本地资源和云端资源入口
- 材料新建、编辑、删除
- 音频绑定和替换
- 分类新增、重命名、删除、移动和排序
- 全部资源、材料搜索
- 听写练习：全部随机、分类随机、练习组随机
- 播放器生命周期管理：离开页面会销毁播放器，重新进入后从头开始
