# 每日英语听写

每日英语听写是一个面向英语学习者的小程序项目，目标是把英文文本和音频材料组织成可播放、可盲听、可循环、可变速、可复听、可记录的听写材料。

当前仓库处于产品和工程规则初始化状态，尚未创建小程序工程、后端服务、数据库、文本转语音接口、内置句子库或词库。

## 产品主线

核心路径：

1. 用户进入系统库、通用听力或自己的用户库。
2. 文本被保存为文本块。
3. 文章自动分句，句子成为播放和音频绑定的基本单位。
4. 用户可以为句子上传 `mp3`、`m4a`、`wav` 音频，也可以在缺音频时使用系统生成音频。
5. 用户通过名称进入材料，避免提前看到英文原文。
6. 用户点击句子播放该句音频，进行盲听、循环、变速和复听。
7. 用户需要时查看原文或图片附件。
8. 系统记录学习时长、听了多少句和重复播放次数。

## 开发规则

本项目采用文档驱动开发。中大型任务、跨文档修改、工程实现和验证闭环必须先维护 `plan.md`。

新增功能时必须保证：

- 代码、测试、文档讲同一个当前事实。
- 失败测试先写清楚。
- 完成后有可复现验证。
- 未验证内容和剩余风险写入收口。
